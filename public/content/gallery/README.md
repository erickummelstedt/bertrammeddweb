# Gallery Content Folder

Place your regular project videos and photo series here.

## For Videos:
- `project-name.mp4`
- `project-name-thumbnail.jpg`

## For Photo Series:
Create a subfolder:
- `series-name/photo-1.jpg`
- `series-name/photo-2.jpg`
- `series-name/photo-3.jpg`
- etc.

Then update `content.json` to reference these files.

See the main README.md for complete instructions!
